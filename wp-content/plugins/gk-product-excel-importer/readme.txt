== Description ==

<p>Import Simple Products for Woocommerce with Excel File on the Fly</p> 
<p>Data mapping by drag and drop excel columns to product attributes.</p>
<p>The form is submitted through AJAX, so there is no page reloading.</p>

= Supports =

* Title
* Description
* Product category
* **Custom Taxonomy for a product (eg. brand)**
* Weight
* Sku
* Regular Price
* Sale Price
* Stock
* Manage Stock
* Instock
* Image

= Extra =

<p> If a subcategory is selected on the excel, parent categories will also be selected for the relevant product. </p>

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory and unzip, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Woo Product Importer Menu Link or Settings link on plugins page to use the plugin.
4. Upload your excel file (there is a link for a sample excel file) and proceed to data mapping and creating your products.
